Joose Tabs Component

Here is a lightweight tabs component.

Please be aware this page will not yet function unless you download two small other assets. This is due to the fact I am wanting to set up node modules of a larger project. This page can be used for reference of the accordion HTML to be used with the CSS and JS assets in this repository.

To get this page to work, you will also need assets from this Joose Component Library which contain utilities such as class manipulation. There are currently dependencies that rely on this, that are not fulfilled in this demo page (they are on the page but are broken links; setup.js and utils.js).

No styling has been applied other than simple show/hide. You can utilise the existing HTML as I have, or you can add your own classes / selectors of your choice. This means you can have horizontally aligned tabs or vertically aligned tabs.

Tabs var componentTabs = new Tabs('#componentContainer', false); or <section id="tabsContainer" data-component="tabs">

The initialisation function will pick out any instances of tabs by searching for the data attribute automatically.
